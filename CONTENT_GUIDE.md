# コンテンツ追加ガイド — Finance for All

学習コンテンツ（コース・カリキュラム・問題・模試）は、**`index.html` を触らずに** `content/` 配下の2ファイルだけで管理できます。

```
finance_e-learning/
├─ index.html              ← UI・画面ロジック（基本さわらない）
└─ content/
   ├─ courses.js           ← コースの構成（概要/ロードマップ/章・単元/問題と模試の一覧）
   └─ questions.js         ← 問題そのもの（設問・選択肢・正解・解説）
```

編集後はブラウザで `index.html` を再読み込みするだけで反映されます（ビルド不要）。

---

## 1. 問題を追加する（`content/questions.js`）

`window.QUESTION_BANKS` は「バンクID → 問題の配列」です。問題1問は次の形：

```js
{
  q: '問題文（HTML可。改行は <br>）',
  choices: ['選択肢A', '選択肢B', '選択肢C', '選択肢D'],
  ans: 1,            // 正解の選択肢インデックス（0始まり。この例ならB）
  exp: '解説文（HTML可）'
}
```

例：`boki3_ch2`（簿記3級・第2章 確認問題）に1問足す場合、その配列の末尾にオブジェクトを追加するだけです。

- 選択肢は2〜6個まで対応（A〜F）。
- バンクを空配列 `[]` にしておくと、画面では「準備中」と表示され、クリックできません（追加待ちの目印）。
- バンクIDは `courses.js` の `bank` と一致させること（下記）。

## 2. コース構成を変える（`content/courses.js`）

`window.COURSES` は「コースID → コース定義」です。主なフィールド：

| フィールド | 意味 |
|---|---|
| `name` / `glyph` / `color` | コース名・アイコン(tabler)・テーマ色(`blue/teal/purple/amber`) |
| `sub` | ヘッダー下の補足（例「3か月 ・ 全32単元 ・ 模擬テスト3回」） |
| `examTitle` / `pass` | 受験名・合格基準の説明 |
| `totalHours` | 想定総学習時間（**学習計画プランナーの計算に使用**） |
| `lead` / `skills` / `targets` | 概要タブの本文・身につくスキル・対象者 |
| `roadmap` | 合格ロードマップの各フェーズ |
| `curriculum` | 章と単元（学習編タブに表示） |
| `quizzes` | 確認問題の一覧（問題編タブ。`bank` で問題バンクに紐付け） |
| `mocks` | 模擬テストの一覧（模擬テストタブ） |

### roadmap の1フェーズ
```js
['active', '2か月目 ・ 第3〜4章', 'タイトル', '説明文', ['動画 18本', '確認問題 60問', 'check:第1回模試'], '学習中']
//  状態      期間ラベル            見出し     本文      タグ（'check:'接頭辞で模試の旗バッジ）        右上バッジ文言
```
状態は `'done' | 'active' | 'todo'`。

### curriculum の1単元
```js
{ id: '1-3', title: '仕訳の基礎', min: 22, status: 'active' }
// status: 'done'（完了） | 'active'（学習中・クリック可） | 'locked'（未開放・クリック不可）
```

### quizzes / mocks の1項目
```js
// 確認問題
{ title: '第1章 確認問題', bank: 'boki3_ch1', status: 'done', score: 80 }
// 模試
{ title: '第1回 模擬テスト', bank: 'boki3_mock1', min: 120, status: 'done', score: 72 }
```
`bank` の文字列が `questions.js` のバンクIDと一致している必要があります。
`status:'locked'` または対応バンクが空のときはクリックできません。

---

## 3. 新しいコースを丸ごと追加する手順

1. `courses.js` に新しいコースID（例 `fp2`）のオブジェクトを追加。
2. そのコースの `quizzes`/`mocks` で使う `bank` を決める（例 `fp2_ch1`）。
3. `questions.js` に同じIDのバンクを作り、問題を入れる。
4. `index.html` のホーム画面「コースを選ぶ」にカードを1枚追加し、`onclick="openCourse('fp2')"` にする。
   （カードのデザインは既存4枚をコピーして文言・色・アイコンを変えるだけ）

> 補足：1〜3だけでコース詳細・問題は動きます。4はホーム一覧に並べるための作業です。将来的にはホームのカードも `courses.js` から自動生成するよう拡張できます。
