# 漫画量産パイプライン — セットアップ手引き（非エンジニア向け）

目標：ComfyUI＋キャラLoRAで「キャラが毎回同じ・フルカラー・本格漫画」を**大量生成**する。
方針：**GPUは自分で設定しない**。まずブラウザで使えるクラウドComfyUIで品質を確認し、
量が見えてから自前GPU購入を検討する。

---

## ステップ0：用語（ざっくり）
- **GPU**：画像生成に必要な計算装置。クラウドで“時間借り”できる。
- **ComfyUI**：生成の手順を「回路（ワークフロー）」で組むツール。一度組めば同設定で連続生成できる。
- **チェックポイント（モデル）**：絵柄の土台。アニメ/漫画系（例：Illustrious / Pony / Animagine 系）を使う。
- **LoRA**：特定キャラを“追加学習”した小さなファイル。これで毎回同じ顔・服にできる。
- **IP-Adapter**：参照画像で顔・雰囲気を寄せる機能（LoRAの補助）。
- **ControlNet**：ポーズや構図を指定する機能。

---

## ステップ1：クラウドComfyUIを借りる（GPU設定不要）
ブラウザでComfyUIが立ち上がる“managed”サービスを使う。代表例（料金・仕様は要確認）：
- **RunComfy** / **ThinkDiffusion** / **MimicPC** … ComfyUIがプリインストール、ブラウザ操作、時間課金
- **RunPod**（ComfyUIテンプレート）… 少し技術寄りだが安価
- **Google Colab**（ComfyUIノート）… 最安で試せる

まずはどれか1つで起動 → ComfyUIの画面が出ればOK（インストール作業なし）。

## ステップ2：絵柄モデルを入れる
- Civitai / Hugging Face から**アニメ・漫画系チェックポイント**を1つ選ぶ（商用利用可のライセンスを確認）。
- managedサービスは「モデル追加」機能から導入。私が候補モデルを提案します。

## ステップ3：キャラLoRAを作る（コード不要ルート）
1. キャラの**一貫した参照画像を10〜20枚**用意（正面/横/表情違い）。
   - 最初の数枚は、確定したキャラ設定プロンプト＋IP-Adapterで“それっぽく”量産→良いものを選別。
2. **Civitaiのオンラインtrainer**（または kohya_ss）に画像を投入してLoRA学習（GPU時間 1〜2時間程度）。
3. できたLoRAをComfyUIに置く → 以後そのキャラが安定して出る。
   ※ さくら／先輩／ムードメーカー先輩／電卓マスコットの**4体分**作るのが理想。

## ステップ4：ワークフローを組む（私が仕様を提供）
ComfyUI上で以下を接続：
`チェックポイント → キャラLoRA → プロンプト（JSONから） → (任意)ControlNetでポーズ → (任意)IP-Adapterで参照 → 生成 → 保存`
- 出力は **「空の吹き出し or 余白」** にして、文字は後工程で乗せる（修正・多言語化が楽）。
- 「API用に保存」しておくと、Pythonから連続生成できる。

## ステップ5：自動化（台本→画像→写植）※私がコードを用意
1. **台本→JSON**：LLMで各コマの「キャラ・表情・ポーズ・背景・セリフ」をJSON化。
2. **JSON→ComfyUI**：APIで流し込み、コマ画像を連続生成。
3. **写植・コマ組み**：Web側オーバーレイ（推奨）または Python(Pillow) で吹き出し＋セリフを合成。
4. **自動QA**：Vision LLMで各コマを採点し、崩れは自動再生成。

---

## まず測る（実装前の“詰め”）
簿記3級 第1章を**1キャラLoRA＋数ページ**で通し、次を記録：
- 1ページの**所要時間**／**GPU時間**（＝コスト）／**手直し率**（キャラ崩れ等）
→ ×2000で全体の費用・期間を見積もり、自前GPU購入の要否を判断。

## 費用の目安（クラウドで試す場合）
- GPU時間：¥80〜300/時（managedサービス）。数ページのテストなら**数百〜数千円**。
- LoRA学習：1キャラ GPU 1〜2時間程度（同上の時間課金内）。
- 量産移行後に枚数が多ければ、**自前GPU（RTX4090級）購入**で追加コストをほぼ電気代化。

## 役割分担
- **あなた / 委託先**：クラウドComfyUIの起動、参照画像の選別、LoRA学習の実行。
- **私（メイン＋frontend/各コンテンツ役）**：候補モデル提案、ワークフロー仕様、キャラ設定＆三面図プロンプト、台本→JSON、写植・Webリーダー、自動QAスクリプト。
- GPU実行はこの環境では行えませんが、繋ぎ込みコードと仕様はすべて用意できます。
